# おはなしのきおく｜小学校受験「お話の記憶」練習アプリ

慶應横浜・桐蔭学園レベルの「お話の記憶」を、ElevenLabs の自然な女性音声で練習できるローカルアプリです。

---

## 帰宅後のセットアップ（3ステップ）

### 1. フォルダを置く
このフォルダ一式を `C:\dev\ohanashi\` に置く。

### 2. APIキーを設定
`.env.example` をコピーして `.env` にリネームし、中身を編集：

```
ANTHROPIC_API_KEY=sk-ant-あなたのキー
ELEVENLABS_API_KEY=sk_あなたのキー
ELEVENLABS_VOICE_ID=21m00Tcm4TlvDq8ikWAM
```

ElevenLabs のキーは notejidou で使っているものと同じでOK。

### 3. 起動
`start.bat` をダブルクリック。
初回だけ自動で必要なライブラリをインストールし、ブラウザが自動で開きます。
（2回目以降は即起動）

---

## 声の選び方（帰宅後に「声くらべ」で決める）

アプリ起動後、トップ画面の「🎙️ 声くらべ」を開くと、**同じお話を複数の声・モデルで一括生成して横並びで聴き比べ**できます。

### 重要：まず「モデル」を理解する
ElevenLabsの日本語は **2025年6月の v3 で品質が大きく向上**しました。お話の記憶では誤読が命取りなので、`eleven_v3` と `eleven_multilingual_v2` の両方を声くらべで比較し、**数字・固有名詞を正しく読む組み合わせ**を選んでください（声くらべは初期で両モデルを比較します）。

### 候補の声
`.env` の `COMPARE_VOICES` に初期登録済み（Rachel / Charlotte / Matilda）。
評判では男性声の **Yamato** が日本語実用的と評価されています。女性声の決定版は固定されていないので、ElevenLabsの Voice Library で「Japanese」「narration」「calm」等で探し、**気になるVoiceIDを `COMPARE_VOICES` に追記**して聴き比べるのが確実です。

書式：`COMPARE_VOICES=表示名:VoiceID,表示名:VoiceID,...`

### 誤読チェック
声くらべのサンプル文は「ふうせんを ふたつ」「みかんを みっつ」「どうぶつは にひき」など**数・色・順序・固有名**を意図的に入れてあります。ここで誤読する声は本番で使わないこと。

### 決まったら
`.env` の `ELEVENLABS_VOICE_ID` と `ELEVENLABS_MODEL` に設定して保存・再起動。

## 機能

- **レベル切替**：慶應横浜（長文・複合記憶・5問）／桐蔭学園（中程度・4問）
- **本番モード**：お話を1回しか再生できないモード。慣れてきたらON
- **声くらべ**：複数の声・モデルを聴き比べて本番の声を決める
- **おでかけパック（オフライン）**：問題＋音声を1つのHTMLに書き出し、iPadでネットなしで練習
- **音声キャッシュ**：同じお話・設問は `audio_cache/` に保存され、2回目以降はクレジットを消費しません
- **クレジット記録**：`usage_log.csv` に文字数・キャッシュ有無が記録されます

---

## おでかけパック（オフラインで持ち歩く）

トップ画面の「🎒 おでかけパックを作る」から。

1. レベルと問題数（最大20、初期10）を選んで「作る」を押す
2. PCが問題と全音声をまとめて生成（数分かかります。進捗バー表示あり）
3. 完成したHTMLファイルをダウンロード
4. **AirDrop / メール / iCloud Drive** などでiPadに送る
5. iPadのSafariかファイルアプリで開く（共有→ホーム画面に追加で“アプリ風”にも）
6. **ネット接続なしで10問** 練習できる

音声をファイルに埋め込むので数MB〜になりますが、iPadなら問題ありません。本番モード（1回再生）もパック内で使えます。
書き出したファイルは `packs/` フォルダにも保存されます。

## 仕組み

```
ブラウザ(タップUI) ─→ Flask(app.py) ─→ Anthropic API（問題文生成）
                                    └→ ElevenLabs API（音声生成＋キャッシュ）
```

APIキーは `.env` にのみ置かれ、ブラウザ側には出ません。

---

## クレジットの目安

ElevenLabs は概ね「1文字＝1クレジット」。お話の記憶1セットは本文+設問で約300〜500文字＝300〜500クレジット程度。
notejidou の 131,000クレジット/月プランなら、キャッシュ込みで毎日数セット練習しても十分余裕があります。
